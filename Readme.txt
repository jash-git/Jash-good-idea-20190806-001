01.腰部伸展(抱膝)
	躺平身體
	單腳左、右和雙腳 抱膝各30秒 為一組
	重複三組

02.臀部伸展
	躺平身體
	單腳呈三角形，另一腿翹腳(儘量打平)
	呈三角形腿執行抱膝維持30秒
	再換腳做
	重複三組

03.橋式
	躺平身體
	雙腳呈三角形
	慢慢將腰部頂到最高點並停留3秒
	重複10次
